# Proyecto Integrado
## Título del proyecto
Plataforma de Gestión de Citas
## Nombre del autor del proyecto
Álvaro De Quinta Rivas
## Descripción
El proyecto consiste en el desarrollo de una plataforma web para la gestión de citas y reservas, enfocada en profesionales independientes como médicos, abogados, terapeutas, entrenadores personales, entre otros. La plataforma permitirá a los usuarios administrar sus citas de manera eficiente y mejorar su productividad, pudiendo añadir a personas involucradas a esa cita (tengan o no cuenta en la plataforma).
## Objetivos
- Facilita la gestión de citas y clientes desde una interfaz intuitiva.

- Reduce la tasa de cancelaciones y ausencias mediante recordatorios automatizados.

- Proporciona un sistema de pago integrado para servicios.

- Permite la sincronización con calendarios externos.

- Un sistema de gestión de citas en línea accesible desde cualquier dispositivo.

- Notificaciones automáticas por telegram para recordar citas.
## Tecnologías utilizadas
Se usará vue para frontend, JWT para gestionar las sesiones, y para backend un conjunto de la framework Slim para PHP, la ORM de Laravel, el cual se llama Eloquent, Docker para desplegar el servicio y la base datos MySQL
